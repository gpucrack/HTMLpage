Qu'est ce que GPUCrack ?

GPUCrack est un casseur de mots de passe Windows. Ce casseur s’appuie sur la technique des compromis temps-mémoire (TMTO) et des rainbow tables. La création de nos tables se fera sur GPU (Graphic processing unit).

Rappel

Pour rappel, la technique des compromis temps-mémoire sépare le travail en deux phases :
  • Construction d’une table à l’aide de calculs de chaînes, c’est la phase offline.
  • Utilisation de la table pour casser les mots de passe, c’est la phase online.
Après avoir réalisé la phase offline une seule fois, le cassage de mots de passe en masse est possible en un temps très court.

Une rainbow table est une structure de données qui permet de retrouver un mot de passe à partir de son hash. Elle se résume à une liste de couples (point de départ, point de fin), où les points de départ et les points de fin sont des mots de passe en clair

Le logiciel phase offline

Cette première application, relative à la phase de pré-calculs génère des rainbow tables, optimise les tables et enfin les exporte. Cette application est éxécutable et distribuable sur GPU.

Le logiciel phase online

Cette seconde application, relative à la phase d’attaque, permet le chargement des rainbow tables en mémoire afin de rechercher le mot de passe correspondant au hash donné. Cette application associé avec une interface graphique permet à l'utilisateur de rechercher un mot de passe de façon ludique ainsi que plusieurs statistiques sur les pré-calculs et la recherche de mots de passe

Le but

Le cassage de mots de passe avec les rainbow tables sur GPU est un sujet de recherche actuel. Des implémentations de rainbow tables sur CPU et GPU ont déjà été réalisées mais il est difficile de trouver un code correct, minimal et documenté. Notre code pourra alors servir de référence dans ce domaine.
Le projet peut apporter de nombreuses réponses sur les différences de performance entre CPU et GPU, sur la possibilité de filtrer nos tables sur GPU ou encore sur la distribution possible de nos tables entre plusieurs GPU.
